编辑器初始化参数说明

参数名 说明 类型 默认值
id 将编辑器渲染到容器的id String ""
initialContent 初始化编辑器的内容 String ""
minFrameHeight 编辑器最小高度 Number 200
textarea 提交表单时，服务器端接收编辑器内容的名字 String "editorValue"
initialStyle 编辑器内部样式 String ""
autoHeightEnabled 是否启用编辑器自动增高属性，true为启用 Boolean 0
enterTag 编辑器回车标签。p或br String "p"
maxUndoCount 最多可以回退的次数 Number 20
maxInputCount 当输入的字符数超过该值时，保存一次现场 Number 20
selectedTdClass 设定选中td的样式名称 String "selectTdClass"
pasteplain 是否纯文本粘贴。false为不使用纯文本粘贴，true为使用纯文本粘贴 Boolean false
focus 初始化时，是否让编辑器获得焦点true或false Boolean false
iframeCssUrl 要引入css的url String ""
serialize 配置过滤器所使用的规则 function null
removeFormatTags 配置格式刷删除的标签 String 'b,big,code,del,dfn, em,font,i,ins,kbd,q,samp, small,span,strike,strong, sub,sup,tt,u,var'
removeFormatAttributes 配置格式刷删除的属性 String 'class,style,lang,width, height,align,hspace,valign'
autoClearinitialContent 是否自动清除编辑器初始内容 Boolean false
indentValue 初始化时，首行缩进距离 String 2em
pageBreakTag 分页符 String _baidu_page_break_tag_
autoFloatEnabled 是否保持toolbar的位置不动 Boolean true
elementPathEnabled 是否启用elementPath Boolean true
wordCount 是否开启字数统计 Boolean true
maximumWords 允许的最大字符数 Number 10000
